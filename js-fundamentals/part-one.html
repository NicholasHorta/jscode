Values & Variables

A value represents the smallest unit of information or data which is of a specific type

A value is chosen by what the you plan to do with it, with brings us to the type of value we choose to use.

JS has TYPED values as apposed to TYPED variables as other languages have, meaning it is our value which represents the type of data we will work with.

  • The built in types are:
  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • Object
  • Symbol

Values stored directly in the source code (code document) are called LITERALS

String literals are strings of information presented between single or double quotations '' or " ":

"String Literal Baby!"

Number literals are simply their value without any form of quotation (Or else it's then seen as a string):

777

Boolean literals, like numbers as presented without any form of quotation (Or else it's then seen as a string):

false
